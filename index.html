Phavorites

Phavorites is a web app that is used to track fan favorites of different song versions by the American rock band Phish. It uses an upvote system and allows for discussion of specific versions. The app was built with TypeScript, Next.JS, Express, MongoDB, and Couldinary. It utilizes the Mongoose ODM, Next-Auth, and Material UI component library.
